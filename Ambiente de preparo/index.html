Ambiente de Preparo

Uma das principais funções do Git são os conceitos do Ambiente de Preparo e do Commit.

Enquanto trabalha, você pode adicionar, editar e remover arquivos. Mas Sempre que você atingir um marco ou terminar uma parte do trabalho, você deve adicionar o arquivos para um ambiente de preparo.

        git add [filename || * || --all || -A]

Git Commit

A adição de commits acompanha nosso progresso e mudanças enquanto trabalhamos. Git considera cada ponto de mudança ou "ponto de salvamento". É um ponto do projeto Você pode voltar se encontrar um bug ou quiser fazer uma alteração.

        git commit -m "message"

Ao adicionar mensagens claras a cada um, é fácil para você (e para os outros) para ver o que mudou e quando.
